Module session_event_stream 

Source
Expand description

Streaming session events for Managed Agents (beta: managed-agents-2026-04-01).

Unlike MessageStream, session events are delivered as complete JSON objects in each SSE data: frame — there is no multi-line content-block reassembly. This stream is therefore self-contained: it accumulates raw bytes, splits SSE frames on blank lines, and deserializes each frame’s concatenated data: payload straight into a SessionEvent.
