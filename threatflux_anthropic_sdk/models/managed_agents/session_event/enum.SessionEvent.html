Expand description

A heterogeneous session event delivered over the events stream or list.

The wire type field selects the variant. Dotted names (agent.message, session.status_idle) cannot be produced by rename_all, so each variant carries an explicit #[serde(rename = "...")]. Unknown event types deserialize to SessionEvent::Unknown so the stream never hard-fails.
