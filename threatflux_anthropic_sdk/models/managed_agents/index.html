Managed Agents data models (beta: managed-agents-2026-04-01).

These models cover the Managed Agents surface: agents, environments, sessions (with events and resources), vaults/credentials, memory stores, and deployments. They mirror the SDK’s existing model conventions — type discriminators via object_type, #[serde(skip_serializing_if)] on options, and a forward-compatible extra: HashMap escape hatch.
